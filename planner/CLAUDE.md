# planner/ conventions

## Filename conventions

### tickets/
Loose ideas and backlog items not yet promoted to an exec plan.
Filename: `YYYY-MM-DD-<short-slug>.md`

### weekly-logs/
Format: `weekly-logs/YYYY/qQ/kwNN/YYYY-MM-DD/protocol.md`

Example: `weekly-logs/2026/q2/kw26/2026-06-25/protocol.md`

Each `protocol.md` contains:
- What was done today
- Decisions made
- Next steps

### annual-logs/
One file per year: `YYYY.md`

### milestones/
One file per milestone reached: `YYYY-MM-DD-<milestone-slug>.md`
